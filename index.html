A new dataset to Monte Carlo noise filtering techniques for Path Tracing

Daker Fernandes Pinheiro, Marcel Santana, Francisco Paulo Magalhães Simões, Verônica Teichrieb

Voxar Labs, Universidade Federal de Pernambuco

Abstract

Photorealistic rendering with global illumination techniques is a computationally expensive task which is usually solved by a stochastic approach. However, these approaches produce Monte Carlo noise at low sampling rates, yielding poor quality images. There are plenty of works which address this problem by using learning based approaches. These techniques usually demand several input data in order to train the algorithm and verify its acuracy but to generate these scenes is a costly task. Therefore, this work presents a new dataset for develop and test algorithms to improve Monte Carlo rendered images. This dataset can be used to test different features and different machine learning approaches, opening this specific denoise problem in the computer graphics field to people from different areas.

Index Terms—monte carlo; denoise; machine learning; deep learning; path tracing; rendering; learned approaches; dataset
